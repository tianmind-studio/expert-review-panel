| 编号 | 严重级 | 位置 | 证据 | 原因 / 风险 | 修改方向 | 认识论标签 |
|------|--------|------|------|-------------|----------|------------|
| 1 | P0 | {具体位置} | {可观察证据 / 原文 / 数据} | {为什么这是问题} | {可执行的修改方向} | [CONSENSUS] |
| 2 | P0 | | | | | |
| 3 | P1 | | | | | |
| ... |  |  |  |  |  |  |

**严重级含义：**
- **P0 · Blocker**：不修则无法提交 / 投稿 / 比赛，作品存在致命伤。
- **P1 · Major**：显著影响评审结果或读者判断，必须修复。
- **P2 · Minor**：润色层级，有时间就改，没时间可接受。

**认识论标签含义：**
- `[CONSENSUS]`：多位专家独立得出的判断。
- `[MAJORITY]`：多数专家支持（非全体）。
- `[MINORITY]`：仅一位专家提出，但主席认为值得保留。
- `[DISPUTED]`：辩论中有实质反驳，存有争议。
- `[⚠️ UNANIMOUS-CHECK]`：全体一致——需警惕共同盲点。
